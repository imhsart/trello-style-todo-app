# 📝 Trello-style Todo Board

This is a small practice project to understand how interactive UI features work in a real-world flow — especially for task management systems like Trello.

It includes creating todos, editing them, auto-deleting empty ones, and moving them across columns using drag-and-drop.

---

## 🚀 What this includes

- Add new **Todo Cards**
- Editable **Todo Content**
- Auto-delete empty Todos (on losing focus)
- Dashboard with **random message generation**
- Different columns for Todo status:
  - To Do
  - In Progress
  - Done
- Drag and Drop functionality between columns

---

## 🧠 What I learned

- Handling dynamic DOM creation
- Using contenteditable for inline editing
- Managing UI cleanup (removing empty elements)
- Implementing drag-and-drop API
- Working with multiple event listeners:
  - drag events
  - focus events
- Improving UX with small details like auto-delete
